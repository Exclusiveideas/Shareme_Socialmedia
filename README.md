## Introduction
This social media web application lies its core interaction with picture upload and interaction. <br/>

It includes:<br/>
User authentication and social login <br/>
Pictures categories<br/>
Account creation<br/>
Comments section<br/>
Profile page <br/>
Similar pictures features<br/>
Posts search<br/>
Responsive design
